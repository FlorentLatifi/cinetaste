# Frontend (React + TypeScript + Vite)

SPA for auth, onboarding (next), For You feed, title detail, watchlist, and search.

## Scripts

```powershell
npm install
npm run dev      # http://localhost:5173
npm run build
```

Vite proxies `/api` → `http://localhost:8000`.
